# Web	alkalmazások	fejlesztése	ASP.NET-el	beadandó feladat	követelményei	
<p><strong>Beadandó feladat</strong></p>
<p>A hallgatónak létre kell hozni egy weboldalt ASP.NET MVC (4 vagy 5) segítségével. View engine-ként a
konzultációkon bemutatott Razor-t (*.cshtml) kell használni. A feladatra kapható maximálisan
megszerezhető pontok száma 100. A létrehozandó weboldal témáját a hallgató szabadon határozhatja meg.</p>

<p><strong>Feladat követelményei</strong></p>
<p>
<ul>
  <li>készítsen saját CSS-t a weboldalhoz</li>
  <li>oldalak száma minimum 7-8</li>
  <li>hozzon létre legalább 3 Controller-t és azon belül minimum 3 action metódust (legyen minimum 2 függvény, ami klienstől visszakapott adatokat kezeli - alapértelmezetten [HttpPost] attribútummal megjelölt, paramétert váró action metódusok</li>
  <li>hozzon létre legalább 2 Model osztályt, és használja őket nézetekben (view)</li>
  <li>nézetek közül 2-3 kérjen be a felhasználótól adatokat (Model osztály, típusos nézet, kapott adatok ellenőrzése)</li>
  <li>nézetek létrehozásakor használjon Html helper függvényeket az alábbi esetekben: form tag, hiperlink, összes bemeneti mező</li>
  <li>használjon Layout-ot (Menüsor, fejléc, lábléc, stb. külön partial view-ként legyen definiálva)</li>
</ul>
</p>

<p><strong>Adatbáziskezelés megvalósítása</strong></p>
<p>
<ul>
  <li>hozzon létre egy 3-4 táblás adatbázist és valósítson meg lekérdezéseket, létrehozásokat és módosításokat</li>
  <li>Entity Framework-öt használjon</li>
  <li>alakítson ki saját design-t (css segítségével) az adatbázisban található elemek megjelenítéséhez (például termékek listázásához)</li>
  <li>létrehozás és módosítás esetén használjon validációt</li>
</ul>
</p>
